# Стоматология Натальи Боровиковой — полный сайт

Сайт развёрнут из утверждённого макета `index.html` в полноценную структуру.
Дизайн-система, шапка, подвал и стили — 1:1 как в одобренном макете.

## Страницы (12)

| Файл | Что это |
|---|---|
| `index.html` | Главная (исходный утверждённый макет + рабочая навигация) |
| `implantaciya.html` | Имплантация — hero, этапы, цены, FAQ |
| `protezirovanie.html` | Протезирование |
| `lechenie.html` | Лечение зубов |
| `gigiena.html` | Профгигиена |
| `brekety.html` | Брекеты и прикус |
| `detskaya.html` | Детская стоматология |
| `doctors.html` | Врачи (3 врача с био) |
| `prices.html` | Полный прайс по 5 категориям |
| `cases.html` | До / После |
| `about.html` | О клинике + принципы |
| `contacts.html` | Контакты + карта + форма записи |

Каждая страница услуги построена по единому сценарию: hero с хлебными
крошками → текстовый блок с фото и преимуществами → 5 этапов лечения →
таблица цен → FAQ → CTA-форма.

## Что уже работает

- **Единая шапка и подвал** на всех страницах, активный пункт меню подсвечивается
- **Мобильное меню** — бургер ниже 860px (выезжающая панель, закрытие по Escape/клику)
- **Навигация** — все ссылки ведут на реальные страницы (не якоря)
- **Формы** — отправляют заявку через `mailto:` (graceful fallback). Поля
  `name/phone/time` + тема письма с указанием страницы
- **WebP** — все фото сконвертированы (3.3 → 0.9 МБ, −74%), отдаются через
  `<picture>` с JPG-фолбеком
- **SEO** — уникальные title/description на каждой странице, sitemap.xml, robots.txt
- **Карта** — Yandex-виджет на странице контактов (координаты центра Ворсмы)

## Что заменить перед публикацией

1. **Форма** — сейчас `mailto:info@bor-dent.ru`. Для нормальной обработки
   подключите Formspree / Tilda Forms / свой бэкенд. В `assets/js/main.js`
   функция отправки в одном месте — заменить там.
2. **Координаты карты** — в `contacts.html` стоит `43.2722,55.9923` (центр
   Ворсмы). Уточнить под точный адрес здания.
3. **Плейсхолдеры `href="#"`** в подвале:
   - «Политика конфиденциальности» — нужна реальная страница/PDF
   - «Акции», «Отзывы» — если разделов не будет, убрать из меню подвала
   - Соцсети VK / IG — вставить реальные ссылки (WhatsApp уже подключён)
4. **Email** — везде `info@bor-dent.ru`, проверьте что почта рабочая.
5. **Фото кейсов** — на `cases.html` и в блоке «До/После» используются
   3 фото по кругу. Заменить на реальные снимки до/после с согласия пациентов.

## Структура

```
borovikova-full/
├── index.html              ← главная (утверждённый макет)
├── implantaciya.html ... contacts.html   ← 11 внутренних страниц
├── sitemap.xml
├── robots.txt
└── assets/
    ├── css/style.css       ← одобренный дизайн + стили внутренних страниц
    ├── js/main.js          ← мобильное меню + обработчик формы
    └── img/                ← JPG + WebP (оба формата, браузер берёт WebP)
```

CSS внутренних страниц дописан в конец `style.css` отдельным блоком
(`/* INNER PAGES */`) — исходные стили макета не тронуты, главная выглядит
ровно как в утверждённой версии.
